# RouteWise-AI

AI-powered urban survival intelligence platform built for navigating Nairobi smarter. Instead of only showing directions, it predicts traffic chaos, fare surges, weather impact, protests, congestion growth, and route reliability to recommend the smartest way to move through the city. Featuring intelligent routing, live map visualization.

## AI-powered urban navigation intelligence for Nairobi

RouteWise is more than a maps app - it is an intelligent mobility copilot designed to help users navigate Nairobi smarter by predicting and adapting to urban chaos in real time.

## The platform combines

- live traffic intelligence
- weather conditions
- congestion prediction
- route reliability
- incident detection
- event disruption awareness
- AI-powered route reasoning

It recommends not just the fastest route, but the smartest one.

## Features

- Intelligent route recommendations
- Alternative route options
- AI-powered route reasoning
- Google Maps integration
- Route rendering
- Alternative route visualization
- Start and destination markers
- Traffic overlays
- Dynamic map fitting
- Chaos Score estimates for disruption severity
- AI commute insights for local travel conditions

## Run Locally

Prerequisites: Node.js

1. Install dependencies:
   npm install
2. Set the GEMINI_API_KEY in .env.local to your Gemini API key.
3. Run the app:
   npm run dev

## Notes

The repository ignores .env files by default, so local secrets stay out of git.
